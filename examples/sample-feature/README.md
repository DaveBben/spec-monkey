# Sample Feature Artifacts

This directory contains example artifacts from the walkthrough in
`../WALKTHROUGH.md`. These are reference outputs — not runnable code.

## Contents

- `research.md` — Phase 0 output (research findings + devil's advocate)
- `impact-map.md` — Phase 1 output (files affected)
- `plan.md` — Phase 2-3 output (approved plan with code snippets)
- `tasks.md` — Phase 4 output (human-readable task list)
- `tasks/plan.json` — Plan-level JSON metadata
- `tasks/task_0.json` — Example task JSON (first task)

These show the expected structure and level of detail for each artifact.
Use them to calibrate what `/feature` should produce.
